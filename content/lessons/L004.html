Lesson 004 – Number Systems

Level: Beginner | Stage: Digital Electronics | Duration: 10–15 minutes

Concept Overview

A number system defines how numbers are represented using a specific set of symbols called digits. In digital electronics, number systems are essential because hardware understands only binary values.

Different number systems are used for human readability, hardware implementation, and compact representation.

Types of Number Systems

  • Decimal: Base 10, digits 0–9
  • Binary: Base 2, digits 0 and 1
  • Octal: Base 8, digits 0–7
  • Hexadecimal: Base 16, digits 0–9 and A–F

Radix (Base)

The radix of a number system is the total number of unique digits it uses. Radix determines the positional value of digits.

Number System Radix
Decimal 10
Binary 2
Octal 8
Hexadecimal 16

Positional Representation

The value of each digit depends on its position and the radix of the number system.

Example:
(1011)2 = 1×2³ + 0×2² + 1×2¹ + 1×2⁰ = 1110

Why Binary is Used

  • Matches transistor ON/OFF states
  • High noise immunity
  • Simpler and reliable circuit design

Key Takeaways

  • Number systems define data representation
  • Binary is the foundation of digital hardware
  • Radix determines positional value
  • Hex and octal simplify binary handling
