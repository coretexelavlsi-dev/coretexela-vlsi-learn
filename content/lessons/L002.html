L002: Logic Gates

Logic gates are the fundamental building blocks of all digital systems. Every processor, controller, and memory device is constructed using combinations of logic gates.

What is a Logic Gate?

A logic gate is a digital circuit that performs a basic logical operation on one or more binary inputs and produces a single binary output.

Inputs and outputs follow binary logic: 0 = LOW / FALSE, 1 = HIGH / TRUE.

Industry View: Logic Gate Symbol Language

In industry and academia, logic gates are represented using standardized symbols. Engineers communicate designs using these symbols instead of equations.

AND Gate

A ───┐
     │
     ├─── AND ─── Y
B ───┘

Y = A · B

Output is HIGH only when both inputs are HIGH.

OR Gate

A ───┐
     │
     ├─── OR ─── Y
B ───┘

Y = A + B

Output is HIGH if any one input is HIGH.

NOT Gate (Inverter)

A ───▷○─── Y

Y = A̅

Output is the logical complement of the input.

Industry View: Truth Table Representation

A  B | AND  OR
------------
0  0 |  0    0
0  1 |  0    1
1  0 |  0    1
1  1 |  1    1

Truth tables are used to verify correctness before hardware implementation.

Where Logic Gates Are Used

  • Arithmetic units (adders, multipliers)
  • Control logic in processors
  • Decision-making circuits
  • Memory address decoding
  • Embedded system controllers
